# Weather Forecaster

​
**Quick Look**

![Image](./assets/images/demo.jpg)
## Description
​
Website for **Weather Forecaster** to give you a quick search of weather for the city.
​
## What it consists of
You are able to enter the city you wish. It will display a 5 day weather report.
